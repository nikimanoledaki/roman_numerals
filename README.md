# Roman Numerals

This algorithm converts integers to roman numerals, with a focus on TDD process and an edge case.

Roman Numerals is an exercise completed during Week 2 of the Makers Academy course.

## Get started
Clone this repo:
```
git clone https://github.com/nikimanoledaki/roman_numerals.git
```

## How to use
```
$ irb -r ./lib/numerals.rb
num = RomanNumeral.new
=> #<RomanNumeral:0x00007fb718878468> 
num.convert(5)
=> "V" 
```

## How to test
```
rspec
```
